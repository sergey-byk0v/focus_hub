# Focus Hub

Chrome extension combining an AM audio modulator and a site blocker with configurable blocking modes.

## Tabs

### Timer
Countdown timer (15m/30m/60m/90m presets or custom). Automatically stops AM modulation when the timer expires.

### Modulation
Amplitude modulation audio processing for any browser tab:
- **Frequency / Depth** — modulation rate and intensity
- **Spatial Movement** — auto-pan left↔right at configurable speed/width
- **Low-Freq Only** — crossover filter, modulates only frequencies below the cutoff
- **Noise** — mix in white/pink/brown/gray noise with optional modulation gating
- **Presets** — save/load custom preset configurations

### Blocking
Two blocking modes for allowed sites:
- **Reason** — intercepts navigation, asks why you're visiting, logs entries with timestamps, CSV export
- **Complete Block** — intercepts navigation, shows "Site Blocked" with a Close Tab button (no logging)

Per-site enable/disable checkboxes, add/remove sites.

### Themes
30 hand-crafted color themes across 6 palettes (dark, slate, medium, light, pastel, vibrant). Unlock new themes through use — one random locked theme becomes available every 12 hours.
